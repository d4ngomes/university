# Sobre
University is a ASP.NET MVC web application for a fictional University using the Entity Framework. It includes functionality such as student admission, course creation, and instructor assignments.

# Technologies Used
## Backend
- C#
- Asp.net MVC
- Identity
- Repository

## Frontend
- HTML / CSS
- Bootstrap
